## Componentes Reutilizáveis

### Kicker
```html
<div class="kicker">Seção</div>
```
- Cor: Accent dinâmico
- Tamanho: 0.65rem
- Linhas decorativas antes e depois

### Headline
```html
<div class="headline">Título com <span class="accent">destaque</span></div>
```
- Tamanho: clamp(2rem, 5vw, 3.8rem)
- Peso: 800
- Responsive com clamp()

### Subline
```html
<div class="subline">Descrição...</div>
```
- Tamanho: clamp(0.9rem, 1.8vw, 1.15rem)
- Cor: Dim (#888)
- Max-width: 680px

### Card
```html
<div class="card glow">Conteúdo</div>
```
- Background: Surface
- Border: Accent com glow
- Padding: 20px 24px

### Steps
```html
<div class="steps">
  <div class="step">
    <div class="step-num">1</div>
    <div class="step-body">
      <div class="step-title">Título</div>
      <div class="step-desc">Descrição</div>
    </div>
  </div>
</div>
```

### Grid Cards
- **Ganha Grid**: 2x2 (responsivo para 1 coluna em mobile)
- **Pilares Grid**: 2x2 (responsivo)
- **Benefit Grid**: 1 coluna (full-width cards)
- **Stats Grid**: 3 colunas (responsivo)

## Navegação & Interatividade

### Keyboard Shortcuts
| Tecla | Ação |
|-------|------|
| ← / → | Navegar slides |
| Espaço | Próximo slide |
| Home | Primeiro slide |
| End | Último slide |
| F | Tela cheia |

### Mouse & Touch
- **Clique nos dots**: Ir para slide específico
- **Scroll**: Navegar
- **Deslize** (mobile): Navegar com gesto

### Indicadores
- **Progress bar**: Barra no rodapé mostra progresso
- **Dots**: Pontos indicam slide atual
- **Counter**: "01 / 16" no canto superior direito
- **Section tag**: Nome da seção no rodapé

## Animações Globais

### Entrada de Elementos
```css
.ai {
  opacity: 0;
  transform: translateY(18px);
  transition: opacity 0.5s ease, transform 0.5s ease;
}
.ai.in {
  opacity: 1;
  transform: translateY(0);
}
```

### Transição de Slides
```css
.slide {
  transition: opacity 0.55s cubic-bezier(.4,0,.2,1),
              transform 0.55s cubic-bezier(.4,0,.2,1);
}
```

### Orbs Flutuantes
- Animação contínua de 10-15 segundos
- Diferentes velocidades e trajetórias
- Blur de 100px para efeito suave

### Particles (Capa)
- 28 partículas subindo
- Duração: 8-18 segundos
- Cores: Rosa, Lima, Azul, Roxo

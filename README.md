# Big Soccer by iSCOUT — Apresentação Interativa

## 📋 Visão Geral

Apresentação profissional interativa para a parceria **Big Soccer + iSCOUT**, desenvolvida com HTML5, CSS3 e JavaScript vanilla. A apresentação possui **16 slides** com navegação fluida, animações suaves e responsividade completa.

---

## 🎯 Objetivo da Apresentação

Apresentar a parceria estratégica entre a **Big Soccer** (escola de futebol consolidada em Atibaia com 20+ anos) e a **iSCOUT** (plataforma de análise de vídeo com IA), focando em:

- ✅ Desenvolvimento humanizado de atletas
- ✅ Análise de dados profunda e responsável
- ✅ Oportunidades para empresas e marcas
- ✅ Alcance e potencial de mídia
- ✅ Segurança e privacidade (LGPD)

- **Espaço** — Próximo slide

- **Clique nos dots** (pontos) — Ir para slide específico

- **F** — Tela cheia
- **Scroll do mouse** — Navegar

### 3. **Responsividade**
- ✅ Desktop (1920px+)

## 🔧 Customizações Possíveis

### Alterar Cores
Edite as variáveis CSS no `<style>`:
```css
:root {
  --brand-pink: #de204c;    /* Cor principal */
  --brand-lime: #c9d148;    /* Cor secundária */
  --slide-accent: #de204c;  /* Accent dinâmico */
}
```

### Adicionar Novos Slides
1. Copie um `<div class="slide">` existente
2. Altere o `data-section` e `data-accent`
3. Atualize o JavaScript se necessário

### Modificar Animações
- Delays: Altere `--d` em cada elemento
- Duração: Modifique `transition` e `animation-duration`
- Easing: Troque `cubic-bezier` nos keyframes

### Trocar Imagens
1. Substitua os arquivos em `/assets/`
2. Mantenha os mesmos nomes ou atualize os `src` no HTML
3. Otimize em WebP para melhor performance
